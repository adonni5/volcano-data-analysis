# Global Volcano Data Analysis

This project explores the global distribution of volcanoes using Python and geospatial tools.

## Objectives

- Analyze volcano distribution worldwide
- Visualize volcano locations using geospatial data
- Identify countries with the highest number of volcanoes
- Explore volcano types and activity status

## Tools

- Python
- Pandas
- GeoPandas
- Matplotlib
- Jupyter Notebook

## Data Source

Global volcano dataset.

## Results

The project includes:

- Global volcano distribution map
- Top countries with most volcanoes
- Most common volcano types
